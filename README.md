📌 Project Overview

This project is a Machine Learning-based Air Quality Index (AQI) Prediction System developed using Python and Scikit-learn.
The system preprocesses air quality data, trains a machine learning model, evaluates its performance, and predicts AQI values for new data.

The project demonstrates a complete end-to-end Machine Learning workflow including:

Data preprocessing
Feature engineering
Pipeline creation
Model training
Cross-validation
Model saving/loading
Prediction generation
🚀 Features

✅ Data Cleaning & Preprocessing
✅ Feature Engineering from Date Column
✅ Numerical & Categorical Pipelines
✅ Missing Value Handling
✅ Standard Scaling
✅ One-Hot Encoding
✅ Random Forest Regressor Model
✅ RMSE & R² Evaluation
✅ Cross Validation
✅ Model Persistence using Joblib
✅ AQI Prediction for New Data
